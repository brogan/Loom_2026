Bezier Draw (Python) — Help

Bezier Draw is a tool for creating and editing three types of geometry that feed into the Loom drawing engine:

  • Closed polygon sets — collections of closed cubic Bézier-curve polygons. Support subdivision, morph targets, and all renderer modes in Loom.
  • Open curve sets — non-closed splines with a distinct start and end. Rendered in Loom as stroked paths or brush stamps; not subdivided.
  • Point sets — collections of discrete anchor positions. Rendered in Loom as dot or brush stamps; not subdivided. Also used as source points for the Mesh Build workflow.

This is the Python (PySide6) reimplementation of the Java Bezier Editor, with all original functionality plus two new features: Mesh Build Mode and a persistent Trace Image layer.

Bezier Draw is normally launched from within the Loom Parameter Editor via the Create / Edit buttons in the Geometry tab. When you close Bezier Draw the editor automatically detects new or changed files. It can also be launched directly from the command line.

Name / Save Panel (below toolbar)

  • Name field — the base filename for saving. Spaces become underscores; non-alphanumeric characters are stripped; the result is lowercased.
  • Save button — writes geometry to the appropriate directories (see Save & Load).
  • Load button — opens a file browser. The root XML element determines what type is loaded.

Canvas (centre)

The drawing grid is 1000 × 1000 user units displayed on a 1040 × 1040 pixel panel (20 px border on each side). The grid has 10 × 10 major divisions. Coordinates are saved in a normalised form centred on 0 (typically −0.5 to +0.5) — the conversion happens automatically on save.

Transform Panel (below canvas)

  • Scale Axis — XY (proportional), X-only, or Y-only.
  • Rotation Axis — Local (each polygon's own centre), Common (shared centroid), or Absolute (canvas centre).
  • Scale slider — drag to scale; release to commit and reset to zero.
  • Rotation slider — drag to rotate (±100 → ±180°); release to commit and reset.
  • Edit Mode checkboxes — Anchor and Control: control which point types are selectable in Point Selection mode.

Drawing Polygons

Entering Create Polygons Mode

Click the Create Polygons toolbar button (or press k). The cursor enters drawing mode for a new polygon.

Placing Points

Each click on the canvas places an anchor point. After the first two anchors, smooth cubic Bézier control points are generated automatically between them. Continue clicking to add more segments.

Closing a Polygon

Click Close Polygon (or ⌘F). The editor connects the last anchor back to the first, records the polygon as finished, and switches to Polygon Selection Mode.

Closed polygons are required for Loom's subdivision pipeline. Use Close Polygon whenever you are creating shapes for subdivision rendering.

Chain-Close from Open Curves

If you have drawn a set of open curves whose endpoints connect to form a closed loop, select all the curves in Open Curve Selection Mode and click Close Polygon. The editor chains them end-to-end (reversing individual curves as needed) and creates one new closed polygon. Endpoint gaps up to 15 px are tolerated.

Open Curves

An open curve is a cubic Bézier spline with a distinct start and end — it does not close back on itself. Used in Loom as stroked paths or brush stamps (not subdivided).

Creating Open Curves

  1. Enter Create Polygons Mode (k).
  2. Place anchor points on the canvas by clicking.
  3. When the curve is complete, click Finish Open Curve instead of Close Polygon. No closing segment is added.

Pressure Sensitivity

When drawing with a pressure-sensitive stylus, Bezier Draw records the stylus pressure at each anchor point. This is saved in the openCurveSet XML as pressure="0.xxx" attributes and used by Loom's BRUSHED renderer to vary stamp size and opacity.

Drawing with a mouse records a uniform pressure of 1.0 at every anchor, producing no variation in Loom. In Point Selection Mode, anchor dot size reflects recorded pressure.
A canvas can contain both closed polygons and open curves, but they are saved as separate files. Keep closed-polygon sessions and open-curve sessions in separate files for clarity.

Creating Point Sets

Point sets are collections of discrete positions. Used in Loom for scatter effects and brush-stamp placement (not subdivided). They are also used as source points for the Mesh Build workflow.

  1. Click Create Points (h) to enter point-placement mode.
  2. Each click on the canvas places a magenta dot.
  3. Click Save. The points are written to pointSets/{name}.xml.
Points have no bezier control points or edges. In Point Selection or Point Creation mode, click within 15 px of a point to select it — it shows as a magenta dot with a yellow ring when selected. Drag to move.

Mesh Build Mode New

Mesh Build Mode lets you construct a closed polygon mesh by picking from an existing point set. You hover over source points to sequence them, then press P to commit each polygon face. All committed faces are accumulated in an auto-created Mesh layer, with adjacent polygon vertices automatically welded at their shared positions.

This is a new capability not present in the Java Bezier Editor. It enables a tracing-based mesh-building workflow: scatter your reference points over a tracing image, then build the mesh topology face by face using Mesh Build Mode.

Typical Workflow — Mesh from Traced Points

  1. Load a trace image — use File → Load Trace Image… to load an image as a persistent trace layer. Position it at the canvas centre as a drawing reference.
  2. Place source points — switch to Create Points mode (h) on your working layer and click to scatter anchor points at key positions on the tracing (mesh vertices, corners, intersections).
  3. Save the point set — save to record your source points in pointSets/.
  4. Activate Mesh Build Mode — click the Mesh toolbar button or press ⌃A (Ctrl+A). The toolbar button highlights in orange. A Mesh layer is created automatically if one does not exist.
  5. Sequence the first face — move the mouse near source points. When the cursor is within 10 px of a point, it is added to the sequence automatically (green numbered ring appears). Lines connect sequenced points in order.
  6. Commit the face — press P when the sequence shows the vertices of one polygon face. A closed quad (or polygon of any count) is built from the sequenced points and added to the Mesh layer. Vertices shared with already-built faces are welded automatically.
  7. Continue building — the sequence clears; mode stays active. Move on to the next face. Adjacent faces share welded anchor points.
  8. Save — click Save. The Mesh layer is written to polygonSets/ as part of a multi-layer .layers.xml bundle. The point set layer is written to pointSets/.

Notes and Tips

  • Source points are not consumed or deleted when used in faces — they remain available for building adjacent faces that share the same vertices.
  • The Mesh layer is created automatically on first commit and cannot be the active drawing layer (only the Mesh Build overlay writes to it). You can still inspect its content by making it active after switching out of Mesh Build Mode.
  • After completing a mesh, use Weld All Adjacent to catch any near-coincident anchors that fall just outside the 5 px auto-weld threshold.
  • For clean subdivision results, build quads (4-point faces) wherever possible.

Trace Image New

A trace image is a background photograph or drawing loaded as a special persistent layer, used as a drawing reference when building geometry. Unlike the session-only reference image, the trace image is saved with the document.

Trace image is a new feature not present in the Java Bezier Editor. The image is saved as a <traceLayer> entry in the .layers.xml manifest, so it reloads automatically the next time the file is opened.

Loading a Trace Image

  1. Choose File → Load Trace Image… from the menu bar.
  2. Select a PNG, JPEG, BMP, TIFF, or WebP image. The browser starts in the project's tracing_images/ folder (created automatically).
  3. The image is loaded as a Trace layer in the Layers panel. It is automatically set as non-active (you cannot draw on it).
  4. The first geometry layer becomes the active layer again.

Behaviour

  • The trace layer appears at the bottom of the canvas at a default opacity of 45%.
  • The layer panel shows it labelled Trace with its visibility checkbox. Toggle visibility to hide/show without unloading.
  • The trace image position, scale, and opacity are stored in the .layers.xml manifest and reload automatically.

Suggested Settings for Mesh Tracing

For the mesh-from-points workflow, load the trace image first, then:

  • The default opacity (~45%) gives a good balance between seeing the image and seeing your geometry. Adjust via the Loom Parameter Editor's trace layer settings if needed.
  • Place your source points (Create Points mode) directly over key structural features: corners, junctions, outline intersections.
  • Keep the trace layer visible while in Mesh Build Mode so you can see which faces you have built relative to the underlying image.

Layers

Every polygon, open curve, and point belongs to exactly one named layer. Only the active layer is selectable; other visible layers appear at 20% opacity as a drawing reference.

Active Layer

Click a row in the Layers panel to make that layer active. The active layer is drawn at full opacity. All new geometry is created on the active layer. Hit-testing is restricted to the active layer.

Layer Buttons

ButtonAction
NewAdd a new layer and make it active.
RenameRename the selected layer (opens a text dialog).
DuplicateCopy all geometry from the selected layer into a new layer. The copy becomes active.
DeleteDelete the selected layer and all its geometry. Cannot delete the last geometry layer. Shows a confirmation warning if the layer has content.
↑ / ↓Move the layer up or down in the draw order (lower = drawn first, underneath).

Multi-layer Export

When the canvas has polygons on two or more layers, clicking Save writes:

  • One {name}_{layerName}.xml per layer.
  • A {name}.layers.xml manifest listing all layers.

If a trace image is loaded, a <traceLayer> element is also written to the manifest so the image path and settings reload automatically.

Use layers to organise complex geometry — for example, source points on one layer and the built mesh on the Mesh layer. Each polygon layer can be given a different renderer in Loom.

Point Selection Mode

Click the Point Selection toolbar button (or press a). Individual anchor and control points can be selected, moved, scaled, and rotated.

Selecting Points — Two-Step Scoping

  1. First click inside a polygon — scopes that polygon (outlined with a yellow dashed border). No points are selected yet.
  2. Click a visible point inside the scoped polygon — selects it.

Clicking directly on a point in any polygon auto-scopes that polygon and selects the point in one step.

Edit Mode Checkboxes

  • Anchor (default: on) — anchor points are selectable and moveable.
  • Control (default: on) — bezier control point handles are selectable and moveable.

Uncheck Control to ignore handles when clicking in dense geometry.

Edge Selection Mode

Click the Edge Selection toolbar button (or press s). A Bézier segment (curve between two consecutive anchor points) is treated as a single selectable edge.

Selecting Edges — Two-Step Scoping

  1. First click inside a polygon — scopes it (yellow dashed outline).
  2. Click within 15 px of a curve — selects that edge.

Edge Highlight Colours

Blue (4 px)Edge selected — Discrete sub-mode
Orange (4 px)Edge selected — Relational sub-mode
Purple (5 px)Edge is weldable, or live duplicate during extrude drag

Operations in Edge Mode

  • Drag — moves the selected edge(s) and their adjacent control points.
  • Duplicate — creates an offset copy of the selected edge as a standalone single-edge open curve.
  • Delete — removes the selected curve segment from its parent polygon.
  • Weld (two edges selected, purple) — snaps both to their midpoint and registers weld links.
  • Extrude — ⇧ Shift + drag (see Extrude).

Open Curve Selection Mode

Click the Open Curve Selection toolbar button (or press d). Entire open (non-closed) curves are selected and moved as units. Closed polygons are not selectable in this mode.

  • Click inside or near an open curve to select it. Outline turns blue (Discrete) or orange (Relational).
  • Clicking a selected curve again deselects it.
  • ⇧ Shift + drag — rubber-band marquee; curves whose centroid falls inside the rectangle are selected.
  • ⌘A — select all open curves on the active layer.
  • With curves selected, clicking Close Polygon attempts to chain them into a new closed polygon.

Polygon Selection Mode

Click the Polygon Selection toolbar button (or press f). Entire closed polygons and ovals are selected and can be dragged, scaled, rotated, flipped, welded, duplicated, or deleted as units.

Selecting Polygons

  • Click inside a polygon to select it. Outline turns blue (Discrete) or orange (Relational).
  • Clicking a selected polygon again deselects it.
  • Click empty canvas to clear all selections.
  • ⌘ + click — add to selection without deselecting.

Drag-to-Weld

While dragging, edges meeting the proximity and alignment thresholds highlight in purple. Release to snap and weld them automatically.

Discrete & Relational Sub-modes

Sub-modeWhat gets selectedWhat moves on drag
DiscreteOnly the directly clicked elementOnly that element — welded partners stay put
Relational (default)Clicked element + all weld-linked elements (transitive)All linked elements — seams remain intact

Switching Sub-mode

  • Tap ⇧ Shift while a selection mode is active → toggles Discrete ↔ Relational. Highlight colours update immediately.
  • ⇧ Shift + click the toolbar selection button when switching modes → activates directly in Discrete.

Scoping

In Point and Edge Selection modes, a scope isolates one polygon so you can click precisely on its points or edges without hitting overlapping polygons. The scoped polygon is outlined with a yellow dashed border (2 px).

  1. Click inside a polygon → it becomes scoped.
  2. Click a point or edge within the scoped polygon → it is selected.
  3. Click inside a different polygon → scope moves there; prior selection cleared (unless ⌘ held).
  4. Click empty canvas → scope and selection both cleared.

Marquee (Rubber-Band) Selection

Hold ⇧ Shift and drag on the canvas to draw a selection rectangle (translucent blue).

  • Point mode — selects all visible points whose position falls inside the rectangle.
  • Edge mode — selects all edges whose anchor-midpoint falls inside the rectangle.
  • Open Curve mode — selects curves whose centroid falls inside the rectangle.
  • Polygon mode — selects closed polygons whose centroid falls inside the rectangle.
When edges are already selected in Edge Mode, ⇧ Shift + drag triggers Extrude instead of the marquee. Deselect all first with ⌘D if you want to marquee in edge mode.

Scale & Rotate

Scale Slider

Drag the Scale slider. Range −100 to +100 maps to a scale factor of 0.0–2.0. Release to commit and reset.

Scale Axis

  • XY (default) — proportional.
  • X only — squeeze/stretch horizontally.
  • Y only — squeeze/stretch vertically.

Scale Pivot

  • Point mode — centroid of selected points.
  • Edge mode — centroid of selected edge anchors.
  • Polygon Relational — shared centroid of all selected polygons.
  • Polygon Discrete — each polygon scales around its own centroid independently.
  • Nothing selected — canvas centre.

Rotate Slider

Drag the Rotation slider. Range −100 to +100 maps to −180° to +180°. Release to commit and reset.

Rotation Axis

  • Local (default) — each polygon rotates about its own centroid.
  • Common — all selected polygons orbit their shared centroid.
  • Absolute — all polygons orbit the canvas centre (500, 500).

Translate (Drag)

  • Point mode — drag any selected point to move all selected points.
  • Edge mode — drag near a selected edge to move all selected edges.
  • Polygon mode — drag inside a selected polygon to move all selected polygons.

An undo snapshot is taken at the start of the first drag movement.

Flip & Centre

Flip Horizontal — mirrors selected geometry left/right around their collective centroid. If nothing is selected, flips all geometry on the active layer.

Flip Vertical — mirrors top/bottom.

Centre — moves the selection (or all geometry if nothing selected) so the group's centroid coincides with canvas centre (500, 500). Relative positions are preserved.

Zoom In / Zoom Out

Zoom InZoom Out
Factor× 1.25× 0.8

Pivot priority: centroid of selected points → centroid of selected edge anchors → centroid of selected polygons → canvas centre.

Zoom operates on coordinate values — it does not change the canvas view. It makes geometry physically larger or smaller. Keep coordinates in the approximate −0.5 to +0.5 normalised range for best compatibility with Loom's sprite sizing system.

Snap to Grid

Grid spacing: 10 units per cell over a 1000 × 1000 canvas (100 × 100 intersections).

Snap Anchors — moves every anchor point on the active layer to the nearest grid intersection. Control points unchanged.

Snap Anchors & Controls — also recalculates control points after snapping, restoring smooth tangents at the new grid positions.

Both operations affect only the active layer and are undoable.

Welding

Welding links anchor points from two different polygons so they share a position and move together in Relational mode. Correct welding is essential for Loom subdivision — unwelded adjacent polygons tear apart when point-level transforms are applied.

Auto-Weld Toggle

The Weld toolbar button is a two-state toggle. When on (default, button checked), drag-to-weld detection runs during polygon drags and edges auto-weld on release. Turn it off when you need to drag a polygon inside another (e.g. before using Intersect) without triggering accidental welds.

Explicit Weld (Edge Mode)

  1. Enter Edge Selection Mode and select exactly two edges from different polygons.
  2. Both conditions must be met: midpoint distance ≤ 80 px and directions within ~45°.
  3. When conditions are met, edges turn purple automatically.
  4. Click the Weld toolbar button.

Both anchor pairs snap to their midpoints; all four point pairs are registered in the Weld Registry.

Weld All Adjacent

Click Weld All Adjacent to find all pairs of edges across different polygons on the active layer whose anchor endpoints are within 5 px of each other, and weld them all in one step.

Extrude

Grows a new closed quad polygon from a selected edge by Shift-dragging. The fastest way to build a repeating grid.

  1. Enter Edge Selection Mode and select one or more edges.
  2. Hold ⇧ Shift and drag on the canvas.
  3. A purple duplicate of the selected edge(s) follows the mouse.
  4. Release at the desired position.

Each extruded edge produces one new closed quad: bottom = original edge, top = dragged position, left/right = straight connectors. If the source edge belongs to a closed polygon, the new quad's bottom anchors are automatically welded to the original edge's anchors. After release, the top edge of the new quad becomes the active selection for immediate chained extrusion.

Use Weld All Adjacent after building a grid to link any unconnected seams.

Intersect (Quad Mesh)

Builds an annular quad-mesh filling the space between two concentric polygons.

  1. Enter Polygon Selection Mode and select exactly 2 polygons (one inside the other).
  2. Click Intersect. Hold ⇧ Shift when clicking to keep the inner polygon; without Shift it is removed.

A ring of quad polygons is created connecting corresponding edge midpoints. All new quads are welded to their neighbours. Works best when both polygons have the same number of sides and are roughly concentric.

Knife Cut

Cuts through polygons along a drawn segment.

  1. Click Knife Tool in the toolbar to enter knife mode.
  2. Drag across the canvas. A dashed red line shows the cut segment.
  3. Release — every polygon on the active layer that the segment physically crosses is split at the intersection points.
  4. Click Knife again (or any selection mode button) to exit knife mode.
The cut is undoable. New polygons at the cut seam are not automatically welded — use Weld All Adjacent afterwards.

Duplicate & Delete

Duplicate

  • Polygon mode — copies of all selected polygons, offset 20 px. Originals are deselected; duplicates become the new selection.
  • Edge mode — a single-edge open curve copy of each selected edge, offset 20 px.

Copy / Paste / Cut

  • ⌘C — copies selected geometry to clipboard (closed polygons, open curves, or edges as open curves).
  • ⌘V — pastes clipboard as new geometry on the active layer, centred on the current mouse position.
  • ⌘X — cut: copies to clipboard then deletes.

Delete

  • Erase Selected Geometry (toolbar) — removes whatever is currently selected (polygons, open curves, edges, or free-standing points).
  • Clear All Geometry — removes all geometry on the canvas. Undoable.

Layer Bundles and the Mesh Layer

When you use Mesh Build Mode, a Mesh layer is added alongside your point layer. On save:

  • pointSets/{name}_{pointLayerName}.xml — the source point set.
  • polygonSets/{name}_mesh.xml — the polygon mesh faces.
  • polygonSets/{name}.layers.xml — manifest listing all polygon layers (and the trace layer if present).

SVG Export & Import

Automatic SVG Export

Every Save writes an SVG copy to svg/{name}.svg. Each closed polygon becomes a <path> using M and C commands in a viewBox="0 0 1000 1000" SVG.

SVG Import

  1. Click Import SVG… from the File menu.
  2. Select an SVG file (browser starts in svg/).
  3. Each <path> is added as new polygon geometry. Existing geometry is preserved.

Supported SVG Path Commands

CommandHandling

Undo

Press ⌘Z to undo the last operation. Up to 20 undo levels are kept. Undo restores complete polygon geometry, all weld links, and layer assignments. The current selection is cleared after an undo.

There is no Redo. If you undo too far, you will need to redo the work manually.
