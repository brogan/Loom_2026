Bezier Editor — Help

The Bezier Editor is a tool for creating and editing three types of geometry that feed into the Loom drawing engine:

  • Closed polygon sets — collections of closed cubic Bézier-curve polygons. Saved as .xml polygon set files. Support subdivision, morph targets, and all renderer modes in Loom.
  • Open curve sets — collections of non-closed cubic Bézier splines. Each curve has a defined start and end; it does not loop back. Saved as .xml open curve set files. Rendered in Loom as stroked paths or brush stamps (no subdivision).
  • Point sets — collections of discrete anchor positions. Saved as .xml point set files. Rendered in Loom as dot or brush stamps (no subdivision).

Polygons can be freely transformed, welded together at shared edges, and extruded to build up complex multi-polygon shapes. The Weld Registry keeps track of which anchor points are linked across polygons so that Relational operations move welded neighbours together automatically.

The editor is organised into layers. Each polygon, curve, or set of points belongs to a named layer. Layers can be shown, hidden, reordered, or renamed independently. Only the active layer is selectable; other visible layers appear at reduced opacity as a drawing reference.

Drawing Polygons

Entering Create Mode

Click the Create Polygon toolbar button (toggle). The cursor enters drawing mode for a new polygon.

Placing Points

Each click on the canvas places an anchor point. After the first two anchors the editor automatically creates smooth cubic Bézier control points between them. Continue clicking to add more segments to the current polygon.

Closing a Polygon

Click the Close Polygon button. The editor:

  1. Connects the last anchor back to the first, forming a closed loop.
  2. Records the polygon's centroid.
  3. Starts a fresh empty polygon manager for the next shape.
  4. Automatically switches to Polygon Selection Mode.
The normal workflow creates closed polygons. To create non-closed open curves for use as an Open Curve Set in Loom, use Finish Open Curve instead of Close Polygon — see the Open Curves section.

Open Curves

An open curve is a cubic Bézier spline with a distinct start and end point — it does not close back on itself. Open curves are used in Loom's Open Curve Sets, which are rendered as stroked paths or brush stamps without subdivision.

Creating Open Curves

  1. Click the Create Polygon toolbar button to enter drawing mode.
  2. Place anchor points on the canvas by clicking. Each click adds a new anchor connected to the previous one by a cubic Bézier segment.
  3. When the curve is complete, click Finish Open Curve (the toolbar button next to Close Polygon). The editor marks the path as open (no closing segment is added) and adds it to the current layer.
  4. A new empty manager is started automatically for the next curve or polygon.
Open curves use the same point and edge selection tools as closed polygons. Scale, rotate, duplicate, and delete all work the same way. However, the Weld, Intersect, and Extrude operations are only meaningful for closed polygons.

Saving as an Open Curve Set

When saving (see Open Curve Set Files), use Save Open Curve Set from the File menu or the save panel. The file is written as an <openCurveSet> XML document (not a polygon set). This format is what the Loom Parameter Editor's Curves sub-tab expects.

Mixed Documents

A single Bezier canvas can contain both closed polygons and open curves, on different layers. When saved as a polygon set, open curves in the file are skipped (only closed curves are exported). When saved as an open curve set, closed polygons are skipped (only open curves are exported). Use layers to keep them organised.

Layers

The Layer Panel appears on the left side of the Bezier Editor window. Every polygon, open curve, and point belongs to exactly one layer. Layers help you organise complex multi-polygon documents and control which geometry is visible and editable at any time.

Layer Panel Layout

The panel contains a table with three columns:

ColumnDescription
Vis (eye icon)Checkbox controlling layer visibility. Hidden layers are not drawn and cannot be selected. Visible non-active layers are drawn at 20% opacity as a reference.
# (number)Layer order number (display only). The stack order affects draw order: lower-numbered layers draw first (underneath).
NameThe layer's display name. Double-click to rename (or use the Rename button).

Active Layer

Click a row in the layer table to make that layer active. The active layer is drawn at full opacity. All new geometry (polygons, open curves) is created on the active layer. Selection hit-tests are restricted to the active layer — you cannot accidentally select geometry on other layers.

Layer Buttons

ButtonAction
NewAdd a new layer above the current active layer. The new layer becomes active.
RenameRename the active layer (opens a text dialog).
DupDuplicate the active layer and all its geometry into a new layer.
DelDelete the active layer and all its geometry. Cannot delete the last remaining layer.
↑ / ↓Move the active layer up or down in the stack. This changes draw order (lower = drawn first, underneath).

Copy / Paste Between Layers

Use ⌘ Cmd+C to copy the selected geometry on the active layer to the clipboard, then switch to a different layer and use ⌘ Cmd+V to paste it onto that layer.

Export Behaviour

When you save a polygon set or open curve set, each layer is exported to a separate file ({baseName}_{layerName}.xml). A layer manifest file ({baseName}.layers.xml) is also written, listing all layers in order. The Loom Parameter Editor reads the manifest and treats the set as a layered bundle.

Single-layer documents save as a standard polygon set or open curve set file (no manifest). The manifest is only written when two or more layers exist.

Point Selection Mode

Click the Point Selection toolbar button to enter this mode. Individual anchor and control points can be selected, moved, scaled, and rotated.

Selecting Points — Two-Step Scoping

  1. First click inside a polygon — scopes that polygon (its outline flashes a yellow dashed border). No points are selected yet.
  2. Click a visible point inside the scoped polygon — selects it.

If you click directly on a point in any polygon (scoped or not), that polygon is automatically scoped and the point selected in one step.

Deselecting / Changing Scope

  • Click empty canvas — clears scope and selection.
  • Click inside a different polygon — changes scope; selection is cleared (unless ⌘ Cmd is held).

Additive Selection

Hold ⌘ Cmd while clicking to add points to the current selection without clearing previous ones.

Dragging Selected Points

Drag any selected point; all selected points (including weld-linked partners if in Relational sub-mode) move together.

Selection Colours

  • Blue — anchor point selected (Discrete sub-mode)
  • White — control point selected (Discrete)
  • Red — anchor point selected (Relational sub-mode)
  • Yellow — control point selected (Relational)

Edge Selection Mode

Click the Edge Selection toolbar button. A Bézier segment (the curve between two consecutive anchor points) is treated as a single selectable edge.

Selecting Edges — Two-Step Scoping

  1. First click inside a polygon — scopes it (yellow dashed outline).
  2. Click within 15 px of a curve inside the scoped polygon — selects that edge.

Clicking directly on a curve in any polygon selects it and automatically scopes that polygon.

Edge Highlight Colours

  • Blue (4 px) — edge selected (Discrete)
  • Orange (4 px) — edge selected (Relational)
  • Purple (5 px) — edge is weldable or being extruded

Dragging Selected Edges

Drag any selected edge to translate it. All four control points of each selected curve move. In Relational sub-mode, weld-linked edges in neighbouring polygons move simultaneously.

Operations Available in Edge Mode

  • Duplicate — creates an offset copy of the selected edge (single-edge polygon), offset 20 px.
  • Delete — removes the selected curve from its parent polygon.
  • Weld — merges exactly two selected edges (must be close and parallel; see Welding).
  • Extrude — ⇧ Shift + drag (see Extrude).

Polygon Selection Mode

Click the Polygon Selection toolbar button. Entire polygons are selected and can be dragged, scaled, rotated, flipped, welded, duplicated, or deleted as units.

Selecting Polygons

  • Click inside a polygon to select it. Its outline is drawn in blue (Discrete) or orange (Relational).
  • Clicking a selected polygon again deselects it.
  • Click empty canvas to clear all selections.

Click-Through for Stacked Polygons

When polygons overlap, clicking inside a selected inner polygon near an unselected outer polygon uses a deferred selection mechanism:

  • Click + drag — moves only the inner polygon (the outer is never added to the selection). This lets you reposition the inner polygon independently.
  • Click + release (no drag) — on mouse release the outer polygon is added to the selection. Both polygons are now selected.

This behaviour is essential for the Intersect workflow: duplicate a polygon, scale it down, drag it inside the original, then click anywhere in the outer region to select both, and finally click Intersect.

Dragging Polygons

Drag a selected polygon to move it. In Relational sub-mode, all polygons linked via welded anchor points move together, keeping seams intact.

Drag-to-Weld

While dragging in Polygon Selection Mode the editor continuously checks for nearby weldable edges on unselected polygons. When a pair of edges meets the proximity and alignment thresholds, they are highlighted in purple. Release the mouse to snap those edges together and register their anchor links — see Welding.

Operations Available in Polygon Mode

  • Duplicate — creates a copy of all selected polygons, offset 20 px.
  • Delete — removes all selected polygon managers.
  • Flip Horizontal / Vertical — mirrors selected (or all) polygons around their collective centroid.
  • Centre — moves selected (or all) polygons to the canvas centre.
  • Scale / Rotate sliders — transform selected (or all) polygons.

Discrete & Relational Sub-modes

Every selection mode has two sub-modes that control how selections propagate through the Weld Registry.

Sub-modeWhat gets selectedWhat moves / transforms
Discrete Only the directly clicked element (point, edge, or polygon) Only the selected element — welded partners stay put. Welds may stretch.
Relational Clicked element + all weld-linked elements, traversed transitively through the Weld Registry All linked elements move together — seams are maintained.

Switching Sub-mode

  • At mode activation time: hold ⇧ Shift when clicking a toolbar selection button to activate in Discrete mode (default is Relational).
  • While a mode is active: tap ⇧ Shift to toggle between Discrete and Relational. The highlight colours update instantly to reflect the current sub-mode.
In Point Relational mode, clicking an anchor point also selects its adjacent control points, keeping the curve shape intact during moves.

Scoping

In Point and Edge Selection modes, a scope isolates one polygon so you can click precisely on its points or edges without accidentally hitting overlapping polygons.

Visual Indicator

The scoped polygon is outlined with a yellow dashed border (2 px) using an 8/4 dash pattern.

Scoping Workflow

  1. Click inside a polygon → it becomes scoped (no selection change yet).
  2. Click on a point or edge within the scoped polygon → it is selected.
  3. Click inside a different polygon → scope moves to the new polygon; prior selection is cleared (unless ⌘ Cmd is held).
  4. Click empty canvas → scope and selection are both cleared.

If you click directly on a point or edge in any unscoped polygon, it is auto-scoped and selected simultaneously in one step.

Marquee (Rubber-Band) Selection

Hold ⇧ Shift and drag on the canvas to draw a selection rectangle.

  • Point mode — selects all visible points whose position falls inside the rectangle.
  • Edge mode — selects all edges whose anchor-midpoint falls inside the rectangle.
  • Polygon mode — selects all polygons whose centroid falls inside the rectangle.

The marquee is drawn as a semi-transparent light-blue fill with a dashed blue stroke. Release the mouse to finalise the selection.

When edges are already selected in Edge Selection Mode, ⇧ Shift + drag triggers Extrude instead of the marquee.

Scale & Rotate

Scale Slider

Drag the Scale slider below the canvas. The range is −100 to +100, mapping to a scale factor of 0.0 to 2.0 (0 = original size).

Scale Axis

  • XY (default) — proportional scale in both axes.
  • X only — squeeze/stretch horizontally only.
  • Y only — squeeze/stretch vertically only.

Scale Pivot

  • In Point mode — centroid of selected points.
  • In Edge mode — centroid of selected edge anchors.
  • In Polygon mode, Relational — single shared centroid of all selected polygons.
  • In Polygon mode, Discrete — each polygon scales around its own centroid independently.
  • Nothing selected — all polygons scale around the canvas centre.

Rotate Slider

Drag the Rotation slider. Range −100 to +100 maps to −180° to +180°.

Rotation Axis (radio buttons)

  • Local (default) — each polygon rotates about its own centroid.
  • Common — all selected polygons orbit their shared centroid.
  • Absolute — all polygons orbit the canvas centre (500, 500).

Slider Commit

Releasing the mouse on either slider commits the transformation (setOrigPosToPos), resets the slider to zero, and marks a new undo point. Subsequent slider drags start fresh from the committed positions.

Translate (Drag)

Elements are translated simply by dragging them on the canvas:

  • Point mode — drag any selected point to move all selected points.
  • Edge mode — drag near a selected edge curve to move all selected edges.
  • Polygon mode — drag inside a selected polygon to move all selected polygons.

In all cases an undo snapshot is taken at the start of the first drag movement.

Flip & Centre

Flip Horizontal

Mirrors selected polygons left/right around their collective centroid. If nothing is selected, all polygons are flipped. Axis: vertical line through the group centroid.

Flip Vertical

Mirrors selected polygons top/bottom around their collective centroid.

Centre

Moves the selected polygons (or all polygons if nothing is selected) so their collective centroid coincides with the canvas centre (500, 500). Welded points move together.

Zoom In / Zoom Out

The Zoom In and Zoom Out toolbar buttons scale all polygon geometry around a pivot point:

Zoom factorInOut
Factor× 1.25× 0.8

Pivot Priority

  1. Centroid of selected points (if any).
  2. Centroid of selected edge anchor pairs.
  3. Centroid of selected polygon centres.
  4. Canvas centre (default when nothing is selected).

Each zoom click is a separate undoable step.

Zoom operates on the coordinate values of all geometry — it does not change the canvas view. Use it to scale the entire polygon set up or down.

Snap to Grid

The grid has 100 × 100 intersections over the 1000 × 1000 canvas (spacing: 10 units per cell).

Snap Anchor Points

The Snap Anchors button moves every anchor point in all polygons to the nearest grid intersection. Control points are left unchanged.

Snap Anchors & Control Points

The Snap Anchors & Controls button also recalculates control points (resetControlPoints()) after snapping anchors, maintaining smooth tangents at the new grid positions.

Both operations are undoable.

Welding

Welding links anchor points from two different polygons so they share a position and move together in Relational mode.

Weld Button — Toggle Mode

The Weld toolbar button is a two-state toggle:

StateIconMeaning
Selected / pressedweld (active)Auto-weld is ON (default). Drag-to-weld detection runs during polygon drags; edges auto-weld on release.
Not selectedweld (normal)Auto-weld is OFF. No drag-weld highlighting; edges do not snap on release.
Disable auto-weld when you need to position a polygon inside another (for example, before using Intersect) without triggering accidental welds during the drag. Re-enable it afterwards for normal assembly work.

Explicit Weld (Edge Mode)

When you are in Edge Selection Mode with exactly two edges selected, clicking the Weld button performs a one-time manual weld and immediately reverts the toggle back to its previous state — it does not change the auto-weld setting.

  1. Enter Edge Selection Mode.
  2. Select exactly two edges (one from each polygon you want to connect).
  3. The edges must satisfy both:
    • Midpoint distance ≤ 80 px
    • Directions within ~45° of parallel (|cos θ| > 0.7)
  4. When both conditions are met the edges turn purple automatically.
  5. Click the Weld toolbar button.

What Happens

  • Both anchor pairs snap to their midpoints.
  • Both control-point pairs snap to their midpoints.
  • All four point pairs are registered in the Weld Registry.
  • The direction pairing (normal vs. reversed) is chosen automatically to minimise total anchor displacement.

Drag-to-Weld (Polygon Mode)

While dragging in Polygon Selection Mode with auto-weld ON, the editor continuously checks for weld candidates:

  1. As you drag a polygon near another, edges that meet the 80 px / 45° criteria turn purple on both polygons.
  2. Release the mouse — the highlighted edge pairs are welded automatically (snap to midpoints, register links).

This allows rapid assembly of multi-polygon shapes by simply dragging pieces together. Turn auto-weld OFF when you want to drag without welding (e.g. positioning the inner polygon for Intersect).

Weld Registry & Relational Mode

The Weld Registry is a bidirectional map of linked anchor pairs. In any Relational sub-mode, operations (drag, scale, rotate) traverse this map transitively — moving one point moves all points it is linked to, and all points those are linked to, and so on.

Knife Cut

The Knife Cut toolbar button is a toggle tool that slices one or more polygons along a straight line you draw across the canvas. It operates independently of the selection modes — activating it suspends the current mode and restores it when deactivated.

Using the Knife

  1. Click the Knife Cut toolbar button to activate it. The current selection mode buttons deselect visually.
  2. Press and drag across the canvas to define the cut line. A red dashed line extends across the full canvas to show the cut direction; small circles mark the drag start and current cursor position.
  3. Release the mouse. Every polygon that the line crosses with an even number ≥ 2 intersections is split into pieces at those intersection points.
  4. Click the Knife Cut button again (or click any selection mode button) to deactivate. The previously active selection mode is restored automatically.

How Cutting Works

The cut uses de Casteljau recursive subdivision on signed distances to find exactly where the cut line crosses each cubic Bézier curve. This preserves the true curve shape in each piece — pieces are not approximated with straight edges.

  • Even intersections (2, 4, 6 …) — the polygon is split into that many pieces. Each consecutive pair of intersection points defines one piece's boundary, with a straight closing edge added across the cut.
  • Odd intersections or zero — the polygon is not cut (a partial hit with no exit is ignored).
  • New pieces inherit selection state — if the original polygon was selected before Knife mode was entered, all pieces from it are added to the selection.

Auto-Weld on Cut

After cutting, coincident anchor points on adjacent pieces are automatically welded (proximity scan, ε = 0.1 px). This keeps cut seams connected so pieces move together in Relational mode. Auto-weld on cut is controlled by the same auto-weld toggle as drag-to-weld — if you have disabled auto-weld, cut pieces will not be welded at their seams.

Undo

The entire cut gesture is a single undoable step. Undo restores all original polygons and removes all pieces.

The minimum drag distance to trigger a cut is 5 px. A brief click without dragging is ignored.

Intersect

The Intersect toolbar button builds an annular mesh between two concentric equivalent polygons — one inside the other — filling the ring-shaped gap between them with N quad polygons (one per side).

Requirements

Before clicking Intersect, select exactly two polygons in Polygon Selection Mode. They must satisfy all of:

ConditionDetail
Same number of anchor points (curves)Minimum 3. A triangle pair, square pair, hexagon pair, etc.
Rotationally compatibleThe orientation of the first anchor point on each polygon must be within 5° of each other (measured from each polygon's centroid).
One fully inside the otherAll anchor points of the inner polygon must fall within the boundary of the outer polygon.

Procedure

  1. Enter Polygon Selection Mode.
  2. Select the outer polygon.
  3. Click inside the inner polygon to select it too. (If the inner is inside the outer and the outer was already selected, use the click-through deferred selection to add the outer: drag the inner to position it, then click in the outer area and release without dragging.)
  4. Click Intersect.

Result

N quad polygons are created, filling the annular region. Each quad spans:

  • Curve 0 — one outer edge (from the outer polygon's Bézier curve)
  • Curve 1 — right spoke (straight, outer anchor → inner anchor)
  • Curve 2 — the matching inner edge reversed
  • Curve 3 — left spoke (straight, inner anchor → outer anchor, closes the quad)

All coincident spoke endpoints across adjacent quads are automatically welded. The new quads are selected in the current sub-mode colour (orange Relational or blue Discrete). Both original polygons are removed.

Keeping the Inner Polygon

Hold ⇧ Shift while clicking Intersect to retain the inner polygon after the operation. The inner polygon's anchors coincide exactly with the inner row of spoke endpoints — they are auto-welded into the mesh.

Typical Workflow

  1. Draw or create a polygon and close it.
  2. Select it and click Duplicate (a copy appears, selected, offset 20 px).
  3. Use the Scale slider to shrink the duplicate.
  4. Drag the duplicate into the interior of the original. Disable auto-weld first to avoid unwanted welds during the drag.
  5. Click anywhere in the outer polygon's area — the outer polygon is added to the selection on release (click-through deferred selection).
  6. Re-enable auto-weld, then click Intersect.

Undo

Intersect is a single undoable step. Undo removes the quads and restores both originals.

Extrude

Extrude creates a new edge from a selected edge by Shift-dragging, and automatically bridges the two with straight connector edges — the 2D equivalent of 3D edge extrusion.

Procedure

  1. Enter Edge Selection Mode and select one or more edges.
  2. Hold ⇧ Shift and begin dragging on the canvas.
  3. A purple duplicate of the selected edge(s) follows the mouse.
  4. Release the mouse.

Result

For each extruded edge the following is created automatically:

  • New edge — the dragged duplicate, positioned where you released.
  • Left connector — a straight single-edge polygon from original.anchor0 to new.anchor0.
  • Right connector — a straight single-edge polygon from original.anchor1 to new.anchor1.
  • 4 weld links — each connector endpoint welded to the corresponding anchor on the original and new edges.

This creates a quad-cage: the original edge, the new edge, and two straight sides.

Chaining Extrusions

After release the new edge becomes the selected edge. You can immediately ⇧ Shift + drag again to extrude further, building up a chain of connected quads.

Undo

A single undo step covers the entire extrusion gesture (snapshot taken at first drag movement).

Duplicate & Delete

Duplicate

  • Polygon mode — creates copies of all selected polygons, offset 20 px right and down.
  • Edge mode — creates a single-edge polygon copy of the selected edge, offset 20 px.

The duplicates are added immediately after the originals and become the new selection.

Delete Selected

  • Polygon mode — removes the selected polygon managers entirely. Their weld links are also unregistered.
  • Edge mode — removes the selected curves from their parent polygon (the polygon continues with fewer segments).

Clear All Geometry

The Clear All toolbar button removes every polygon from the canvas and resets the weld registry. This is undoable.

Reference Image

A background image can be loaded as a drawing guide:

  1. Click Browse in the Reference Image panel at the top.
  2. Select a PNG, JPEG, or GIF file.
  3. The image is displayed behind the polygon geometry.
  4. Toggle its visibility with the Toggle Reference Image toolbar button.
The reference image is not saved with the polygon set — it is a session-only drawing aid.

Save & Load Polygon Sets

Saving

  1. Type a name in the Name field in the top panel.
  2. Click Save Polygons.

The polygon set is written as <name>.xml into the project's polygonSets/ directory. An SVG copy is also written automatically to an adjacent svgs/ directory. Only closed curves are included; any open curves on the canvas are skipped.

Loading

  1. Click Load to open a file browser, or type the path in the File field.
  2. Select an XML polygon set file.
  3. The geometry is loaded onto the canvas, replacing the current content. The Name field is populated automatically.
Loading a polygon set replaces all current geometry. Use Undo if you load by mistake.

Open Curve Set Files

Open curve sets use a different XML format (<openCurveSet> root element) from regular polygon sets (<polygonSet> root). The editor automatically detects which format to use based on whether the canvas contains open or closed curves when you save.

Saving an Open Curve Set

  1. Ensure your canvas contains open curves (created with Finish Open Curve).
  2. Type a name in the Name field.
  3. Click Save Open Curves (or use File → Save as Open Curve Set). The file is written to the curveSets/ directory of the current project.

Loading an Open Curve Set

Use Load and select an .xml file whose root element is <openCurveSet>. The editor detects the format automatically and loads the geometry as open curves.

The Loom Parameter Editor's Edit in Bezier button in the Curves sub-tab passes the correct headers to Bezier Draw automatically. You do not normally need to load open curve set files manually.

Morph Target Files

Morph animation in Loom blends a sprite's base shape toward one or more morph target shapes. You create morph targets by drawing an alternate version of the base geometry in Bezier Draw, then saving it to the project's morphTargets/ directory. Multiple morph targets form a chain: base → mt1 → mt2 → …

File naming conventions

Files in morphTargets/ use a suffix to indicate their geometry type:

SuffixGeometry typeExample
.poly.xmlClosed polygon set (<polygonSet> root)myshape_mt_1.poly.xml
.curve.xmlOpen curve set (<openCurveSet> root)myshape_mt_1.curve.xml

The suffix determines how Loom loads the file. Both types can coexist in the same morphTargets/ directory. Legacy files with no recognised suffix are treated as polygon format for backward compatibility.

Topology rule: each morph target must have the same number of polygons/curves, the same number of vertices per polygon/curve, and the same polygon type (SPLINE or LINE) as the base shape. Loom detects mismatches at load time and falls back to the base shape.

Step-by-step: creating a polygon morph target

  1. In Bezier Draw, open or draw the base polygon set that you want to morph from.
  2. Make a note of the polygon count and vertices. You will need to preserve these exactly in every morph target.
  3. Edit the polygon shapes to produce the alternate pose (the target you want to morph to). Do not add, remove, or split polygons, and do not add or remove anchor points.
  4. In the Name field type the target filename, e.g. myshape_mt_1.
  5. Click Save Polygons. The file is saved as myshape_mt_1.xml in polygonSets/. Move and rename it to morphTargets/myshape_mt_1.poly.xml manually (or use the Parameter Editor's morph target panel to create the initial copy automatically).
  6. To create a second target (_mt_2), repeat from a copy of the base and save as myshape_mt_2.poly.xml.

Step-by-step: creating an open curve morph target

  1. Open or draw the base open curve set. Note the curve count and vertex count per curve.
  2. Edit the curve shapes to the desired target pose without adding or removing curves or anchor points.
  3. Click Save Open Curves. The file is written to curveSets/.
  4. Move and rename it to morphTargets/myshape_mt_1.curve.xml.

Layer bundles and morph targets

If the base polygon set uses multiple layers (saved as a .layers.xml bundle), each morph target must be saved in the same bundle format with the same layer structure. The Parameter Editor handles this automatically when you use its Edit in Bezier workflow — edit, save, and close Bezier; the editor re-imports the result.

The safest workflow: always start from a copy of the base polygon set file when creating morph targets. This guarantees the initial topology matches exactly. Only then move the vertices to the target pose.

SVG Export & Import

Automatic SVG Export

Every time you click Save, an SVG copy of the polygon set is saved alongside the XML in the svgs/ directory (adjacent to polygonSets/). No extra action required.

Each closed polygon becomes a <path> element using M and C commands in a viewBox="0 0 1000 1000" SVG. Stroke is black, fill is none.

SVG Import

  1. Click Import SVG in the SVG panel.
  2. Select any SVG file. The browser opens in the svgs/ directory by default.
  3. Each <path> in the SVG is added to the canvas as a new polygon. Existing geometry is preserved.

Supported SVG Path Commands

CommandHandling
M / mMove to (subpath start). Subsequent pairs treated as implicit line-tos.
C / cCubic Bézier — used directly.
L / lLine to — converted to a cubic with control points at ⅓ and ⅔.
Q / qQuadratic Bézier — converted to cubic via degree elevation.
Z / zClose path — adds a closing line segment if not already closed.
H, V, S, T, A, …Silently skipped.

All paths are treated as closed polygons. Coordinates are mapped from the SVG viewBox into Bezier canvas space automatically.

Importing is undoable with a single undo step.

Undo

Press ⌘ Cmd + Z to undo the last operation.

  • Up to 20 undo levels are kept (older states are discarded).
  • Undo restores the complete polygon geometry (all anchor and control point positions) and all weld links.
  • Selection is cleared after an undo.

Operations that create undo points

  • Any drag start (point, edge, or polygon drag)
  • First movement of a Scale or Rotate slider gesture
  • Close Polygon
  • Clear All Geometry
  • Zoom In / Zoom Out (each click)
  • Snap Anchors / Snap All
  • Flip, Centre
  • Weld (explicit and drag-to-weld)
  • Knife Cut (single step covers the entire cut gesture)
  • Intersect
  • Extrude (start of drag)
  • Duplicate, Delete
  • SVG Import
There is no Redo. If you undo too far, you will need to redo the work manually.
