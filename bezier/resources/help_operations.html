Operations

Scale & Rotate

Scale Slider

Drag the Scale slider below the canvas. Range −100 to +100 maps to a scale factor of 0.0–2.0. Releasing the mouse commits the transformation, resets the slider to zero, and marks an undo point.

Scale Axis

  • XY (default) — proportional scale in both axes.
  • X only — squeeze/stretch horizontally only.
  • Y only — squeeze/stretch vertically only.

Scale Pivot

  • Point mode — centroid of selected points.
  • Edge mode — centroid of selected edge anchors.
  • Polygon Relational — shared centroid of all selected polygons.
  • Polygon Discrete — each polygon scales around its own centroid independently.
  • Nothing selected — canvas centre.

Rotate Slider

Drag the Rotation slider. Range −100 to +100 maps to −180° to +180°. Release to commit and reset.

Rotation Axis

  • Local (default) — each polygon rotates about its own centroid.
  • Common — all selected polygons orbit their shared centroid (good for circular arrangements).
  • Absolute — all polygons orbit the canvas centre (500, 500).

Translate (Drag)

  • Point mode — drag any selected point to move all selected points.
  • Edge mode — drag near a selected edge to move all selected edges and their control points.
  • Polygon mode — drag inside a selected polygon to move all selected polygons.

An undo snapshot is taken at the start of the first drag movement.

Flip & Centre

Flip Horizontal — mirrors selected geometry left/right around the collective centroid of the selection. If nothing is selected, flips all geometry on the active layer (closed polygons, open curves, and ovals).

Flip Vertical — mirrors selected geometry top/bottom around the collective centroid.

Centre — moves the selection (or, if nothing is selected, all geometry on the active layer) so the group’s collective centroid coincides with the canvas centre (500, 500). The entire group translates together as a unit — relative positions are preserved.

Zoom In / Zoom Out

The Zoom toolbar buttons scale all polygon geometry on the active layer around a pivot point:

InOut
Factor×1.25×0.8

Pivot Priority

  1. Centroid of selected points.
  2. Centroid of selected edge anchor pairs.
  3. Centroid of selected polygon centres.
  4. Canvas centre (default when nothing is selected).
Zoom operates on coordinate values — it does not change the canvas view. It makes the geometry physically larger or smaller on the canvas. Keep polygon coordinates in the approximate −0.5 to +0.5 normalised range for best compatibility with Loom's sprite sizing system.

Snap to Grid

The grid has 100×100 intersections over the 1000×1000 canvas (10 units per cell). Both snap operations act only on the active layer.

Snap Anchors — moves every anchor point on the active layer to the nearest grid intersection. Control points are left unchanged.

Snap Anchors & Controls — also recalculates control points after snapping anchors, restoring smooth tangents at the new grid positions. Use this to clean up handles that have drifted into unusual positions.

Copy, Paste & Cut

⌘ + C — copies the current selection to an internal clipboard:

  • Polygon Selection mode — copies selected closed polygons and ovals.
  • Open Curve Selection mode — copies selected open curves.
  • Edge Selection mode — copies each selected edge as a standalone single-edge open curve.
  • Point Selection mode — no-op (individual control points have no paste target).

⌘ + V — pastes the clipboard as new geometry on the active layer. The pasted shapes are centred on the current mouse cursor position — hover the mouse where you want the paste to land before pressing Cmd+V. An undo snapshot is taken.

⌘ + X — Cut: copies the selection to the clipboard (same as Cmd+C) then immediately deletes it.

Paste always targets the active layer regardless of which layer the original geometry was on.

Welding

Welding links anchor points from two different polygons so they share a position and move together in Relational mode. Proper welding is essential when building polygon meshes for Loom — unwelded adjacent polygons will tear apart when point-level transforms are applied.

Explicit Weld (Edge Mode)

  1. Enter Edge Selection Mode and select exactly two edges — one from each of two different polygons.
  2. Both conditions must be met: midpoint distance ≤ 80 px and directions within ~45° of parallel (|cosθ| > 0.7).
  3. When conditions are met, both edges turn purple automatically.
  4. Click the Weld toolbar button.

Both anchor pairs snap to their midpoints, control-point pairs snap to their midpoints, and all four point pairs are registered in the Weld Registry.

Drag-to-Weld (Polygon Mode)

While dragging a polygon in Polygon Selection Mode, edges meeting the 80 px / 45° criteria turn purple. Release the mouse to weld them automatically. Auto-weld can be toggled on/off via the Weld toolbar button.

Weld All Adjacent

Click Weld All Adjacent to find all pairs of edges across different polygons on the active layer whose anchor endpoints are within 5 px of each other, and weld them all in one step. Useful after placing or importing several polygons that should share boundaries.

Weld Registry & Relational Mode

The Weld Registry is a bidirectional map of linked anchor pairs. In Relational sub-mode, moving one point moves all transitively linked points — so a whole mesh moves cleanly when you drag any part of it.

Extrude

Extrude grows a new closed quad polygon from a selected edge by Shift-dragging. This is the fastest way to build a repeating grid or mesh of connected polygons.

Procedure

  1. Enter Edge Selection Mode and select one or more edges.
  2. Hold ⇧ Shift and begin dragging on the canvas.
  3. A purple duplicate of the selected edge(s) follows the mouse.
  4. Release the mouse at the desired position.

Result

Each extruded edge produces one new closed quad polygon with four sides:

  • Bottom — the original selected edge (left in place, unchanged).
  • Top — the dragged edge at the release position (curve reversed so the polygon winds correctly).
  • Left & Right connectors — straight edges bridging the corresponding anchor pairs.

Welding

If the source edge belongs to a closed polygon, the new quad’s bottom anchors are automatically welded to the original edge’s anchors — the shared seam is immediately linked in the Weld Registry. If the source edge belongs to an open curve, no weld is created (the new quad is standalone geometry).

After release, the top edge of the new quad becomes the active selection, so you can immediately ⇧ Shift + drag again to chain further extrusions.

Use Weld All Adjacent after building a grid to automatically link any unconnected seams between quads.

Intersect (Quad Mesh)

Builds an annular quad-mesh filling the space between two concentric polygons.

Procedure

  1. Enter Polygon Selection Mode and select exactly 2 polygons (one inside the other).
  2. Click the Intersect toolbar button.
  3. To retain the inner polygon after the operation, hold ⇧ Shift when clicking Intersect. Without Shift, the inner polygon is removed.

Result

A ring of quad polygons is created between the two selected polygons, connecting corresponding edge midpoints around both boundaries. The number of quads equals the number of sides of the outer polygon. All new quads are welded to their neighbours.

Intersect works best when both polygons have the same number of sides and are roughly concentric. The result is a topologically clean quad ring suitable for subdivision in Loom.

Knife Tool

The Knife Tool cuts through one or more polygons along a drawn segment.

Procedure

  1. Click the Knife toolbar button to enter Knife mode (toggle). The previous selection mode is remembered and restored when you deactivate the Knife.
  2. Drag across the canvas. A dashed red line with a perpendicular tick at the start shows the cut segment.
  3. Release the mouse. Every polygon on the active layer that the drag segment physically crosses is split at the intersection points.
  4. Click the Knife button again (or click any selection mode button) to exit Knife mode.
Only polygons that lie between the start and end of the drag are cut — geometry beyond the endpoints is unaffected. This makes it safe to cut one side of a donut or mesh without splitting the opposite side. The cut is undoable. New polygons at the cut seam are not automatically welded — use Weld All Adjacent afterwards if you want the seam linked.

Duplicate & Delete

Duplicate

  • Polygon mode — copies of all selected polygons, offset 20 px right and down. Originals are deselected; duplicates become the new selection.
  • Edge mode — a single-edge open curve copy of each selected edge, offset 20 px. Useful for quickly creating a parallel edge to extrude or weld.

Erase Selected Geometry (⌘ + X or toolbar button)

Removes whatever is currently selected, regardless of type:

  • Polygon mode — removes selected closed polygons and ovals; unregisters their weld links.
  • Open Curve mode — removes selected open curves.
  • Edge mode — removes selected curve segments from their parent polygons.
  • Point mode / Point Selection mode — removes selected free-standing discrete points.

Clear All Geometry — removes all geometry on the canvas and resets the weld registry. Undoable.
