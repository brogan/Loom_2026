Operations

Scale & Rotate

Scale Slider

Drag the Scale slider below the canvas. Range −100 to +100 maps to a scale factor of 0.0–2.0.

Scale Axis

  • XY (default) — proportional scale in both axes.
  • X only — squeeze/stretch horizontally.
  • Y only — squeeze/stretch vertically.

Scale Pivot

  • Point mode — centroid of selected points.
  • Edge mode — centroid of selected edge anchors.
  • Polygon Relational — shared centroid of all selected polygons.
  • Polygon Discrete — each polygon scales around its own centroid independently.
  • Nothing selected — canvas centre.

Rotate Slider

Drag the Rotation slider. Range −100 to +100 maps to −180° to +180°.

Rotation Axis (radio buttons)

  • Local (default) — each polygon rotates about its own centroid.
  • Common — all selected polygons orbit their shared centroid.
  • Absolute — all polygons orbit the canvas centre (500, 500).

Slider Commit

Releasing the mouse commits the transformation, resets the slider to zero, and marks a new undo point.

Translate (Drag)

  • Point mode — drag any selected point to move all selected points.
  • Edge mode — drag near a selected edge to move all selected edges.
  • Polygon mode — drag inside a selected polygon to move all selected polygons.

An undo snapshot is taken at the start of the first drag movement.

Flip & Centre

Flip Horizontal — mirrors selected polygons left/right around their collective centroid. If nothing is selected, all polygons are flipped.

Flip Vertical — mirrors selected polygons top/bottom around their collective centroid.

Centre — moves selected (or all) polygons so their collective centroid coincides with the canvas centre (500, 500).

Zoom In / Zoom Out

The Zoom toolbar buttons scale all polygon geometry around a pivot:

InOut
Factor×1.25×0.8

Pivot Priority

  1. Centroid of selected points.
  2. Centroid of selected edge anchor pairs.
  3. Centroid of selected polygon centres.
  4. Canvas centre (default when nothing is selected).
Zoom operates on coordinate values — it does not change the canvas view. Use it to scale the entire polygon set up or down.

Snap to Grid

Grid has 100×100 intersections over the 1000×1000 canvas (10 units per cell).

Snap Anchors — moves every anchor point to the nearest grid intersection. Control points are left unchanged.

Snap Anchors & Controls — also recalculates control points after snapping anchors, maintaining smooth tangents at the new grid positions.

Welding

Welding links anchor points from two different polygons so they share a position and move together in Relational mode.

Explicit Weld (Edge Mode)

  1. Enter Edge Selection Mode and select exactly two edges (one from each polygon).
  2. Both conditions must be met: midpoint distance ≤ 80 px and directions within ~45° of parallel (|cosθ| > 0.7).
  3. When conditions are met, edges turn purple automatically.
  4. Click the Weld toolbar button.

Both anchor pairs snap to their midpoints, control-point pairs snap to their midpoints, and all four pairs are registered in the Weld Registry.

Drag-to-Weld (Polygon Mode)

While dragging in Polygon Selection Mode, edges meeting the 80 px / 45° criteria turn purple. Release the mouse to weld them automatically.

Weld Registry & Relational Mode

The Weld Registry is a bidirectional map of linked anchor pairs. In Relational sub-mode, moving one point moves all transitively linked points.

Extrude

Extrude creates a new edge from a selected edge by Shift-dragging, bridged by straight connector edges.

Procedure

  1. Enter Edge Selection Mode and select one or more edges.
  2. Hold ⇧ Shift and begin dragging on the canvas.
  3. A purple duplicate of the selected edge(s) follows the mouse.
  4. Release the mouse.

Result

  • New edge — the dragged duplicate at the release position.
  • Left connector — straight edge from original.anchor0 to new.anchor0.
  • Right connector — straight edge from original.anchor1 to new.anchor1.
  • 4 weld links — each connector endpoint welded to the corresponding anchor.

After release, the new edge becomes the selected edge — you can immediately Shift+drag again to chain extrusions.

Duplicate & Delete

Duplicate

  • Polygon mode — copies of all selected polygons, offset 20 px right and down.
  • Edge mode — single-edge polygon copy of the selected edge, offset 20 px.

Delete Selected

  • Polygon mode — removes selected polygon managers and unregisters their weld links.
  • Edge mode — removes selected curves from their parent polygon.

Clear All — removes every polygon and resets the weld registry. Undoable.
