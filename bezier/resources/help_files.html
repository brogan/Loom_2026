Reference Image

  1. Click Browse in the Reference Image panel at the top.
  2. Select a PNG, JPEG, or GIF file.
  3. The image is displayed behind the polygon geometry.
  4. Toggle its visibility with the Toggle Reference Image toolbar button.
The reference image is not saved with the polygon set — it is a session-only drawing aid.

Save & Load Polygon Sets

Saving

  1. Type a name in the Name field in the top panel.
  2. Click Save.

The polygon set is written as <name>.xml into the project's polygonSets/ directory. An SVG copy is also written automatically to an adjacent svgs/ directory.

Loading

  1. Click Load to open a file browser, or type the path in the File field.
  2. Select an XML polygon set file.
  3. The geometry is loaded onto the canvas, replacing the current content. The Name field is populated automatically.
Loading a polygon set replaces all current geometry. Use Undo if you load by mistake.

SVG Export & Import

Automatic SVG Export

Every time you click Save, an SVG copy is saved alongside the XML in the svgs/ directory. Each closed polygon becomes a <path> element using M and C commands in a viewBox="0 0 1000 1000" SVG.

SVG Import

  1. Click Import SVG in the SVG panel.
  2. Select any SVG file. The browser opens in the svgs/ directory by default.
  3. Each <path> in the SVG is added to the canvas as a new polygon. Existing geometry is preserved.

Supported SVG Path Commands

CommandHandling
M / mMove to (subpath start).
C / cCubic Bézier — used directly.
L / lLine to — converted to cubic with control points at ⅓ and ⅔.
Q / qQuadratic Bézier — converted to cubic via degree elevation.
Z / zClose path.
H, V, S, T, A, …Silently skipped.

Coordinates are mapped from the SVG viewBox into Bezier canvas space automatically. Importing is undoable.

Undo

Press ⌘ Cmd + Z to undo the last operation. Up to 20 undo levels are kept.

Undo restores the complete polygon geometry and all weld links. Selection is cleared after an undo.

Operations that create undo points

  • Any drag start (point, edge, or polygon)
  • First movement of a Scale or Rotate slider gesture
  • Close Polygon
  • Clear All Geometry
  • Zoom In / Zoom Out (each click)
  • Snap Anchors / Snap All
  • Flip, Centre
  • Weld (explicit and drag-to-weld)
  • Extrude (start of drag)
  • Duplicate, Delete
  • SVG Import
There is no Redo. If you undo too far, you will need to redo the work manually.
