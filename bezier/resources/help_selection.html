Selection Modes

Point Selection Mode

Click the Point Selection toolbar button. Individual anchor and control points can be selected, moved, scaled, and rotated.

Selecting Points — Two-Step Scoping

  1. First click inside a polygon — scopes that polygon (its outline shows a yellow dashed border). No points are selected yet.
  2. Click a visible point inside the scoped polygon — selects it.

If you click directly on a point in any polygon, that polygon is automatically scoped and the point selected in one step.

Additive Selection

Hold ⌘ Cmd while clicking to add points to the current selection without clearing previous ones.

Edge Selection Mode

Click the Edge Selection toolbar button. A Bézier segment (the curve between two consecutive anchor points) is treated as a single selectable edge.

Selecting Edges — Two-Step Scoping

  1. First click inside a polygon — scopes it (yellow dashed outline).
  2. Click within 15 px of a curve inside the scoped polygon — selects that edge.

Clicking directly on a curve in any polygon selects it and auto-scopes that polygon.

Edge Highlight Colours

Purple (5 px)Edge is weldable or live during extrude drag

Operations in Edge Mode

  • Duplicate — offset copy of the selected edge (20 px).
  • Delete — removes the selected curve from its parent polygon.
  • Weld — merges exactly two selected edges (see Operations tab).
  • Extrude — ⇧ Shift + drag (see Operations tab).

Polygon Selection Mode

Click the Polygon Selection toolbar button. Entire polygons are selected and can be dragged, scaled, rotated, flipped, welded, duplicated, or deleted as units.

Selecting Polygons

  • Click inside a polygon to select it. Its outline is drawn in blue (Discrete) or orange (Relational).
  • Clicking a selected polygon again deselects it.
  • Click empty canvas to clear all selections.

Drag-to-Weld

While dragging in Polygon Selection Mode the editor checks for nearby weldable edges on unselected polygons. When edges meet the proximity and alignment thresholds, they highlight in purple. Release the mouse to snap and weld them.

Discrete & Relational Sub-modes

Sub-modeWhat gets selectedWhat moves
DiscreteOnly the directly clicked elementOnly that element — welded partners stay put
RelationalClicked element + all weld-linked elements (transitively)All linked elements together — seams intact

Switching Sub-mode

  • Hold ⇧ Shift when clicking a toolbar selection button → activates in Discrete mode (default is Relational).
  • While a mode is active: tap ⇧ Shift to toggle between Discrete and Relational. Highlight colours update instantly.
In Point Relational mode, clicking an anchor also selects its adjacent control points, keeping the curve shape intact during moves.

Scoping

In Point and Edge Selection modes, a scope isolates one polygon so you can click precisely on its points or edges without hitting overlapping polygons.

The scoped polygon is outlined with a yellow dashed border (2 px) using an 8/4 dash pattern.

  1. Click inside a polygon → it becomes scoped.
  2. Click on a point or edge within the scoped polygon → it is selected.
  3. Click inside a different polygon → scope moves; prior selection is cleared (unless ⌘ Cmd held).
  4. Click empty canvas → scope and selection both cleared.

Marquee (Rubber-Band) Selection

Hold ⇧ Shift and drag on the canvas to draw a selection rectangle.

  • Point mode — selects all visible points whose position falls inside the rectangle.
  • Edge mode — selects all edges whose anchor-midpoint falls inside the rectangle.
  • Polygon mode — selects all polygons whose centroid falls inside the rectangle.
When edges are already selected in Edge Selection Mode, ⇧ Shift + drag triggers Extrude instead of the marquee.
