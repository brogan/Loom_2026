Selection Modes

Active layer only. All selection and hit-testing operates exclusively within the currently active layer (shown in bold in the Layers panel). Geometry on other layers cannot be selected or moved, even if visible.

Point Selection Mode

Click the Point Selection toolbar button. Individual anchor and control points can be selected, moved, scaled, and rotated.

Selecting Points — Two-Step Scoping

  1. First click inside a polygon — scopes that polygon (its outline shows a yellow dashed border). No points are selected yet.
  2. Click a visible point inside the scoped polygon — selects it.

If you click directly on a point in any polygon, that polygon is automatically scoped and the point selected in one step.

Additive Selection

Hold ⌘ Cmd while clicking a point to add it to the current selection without clearing previous ones.

Edit Mode Checkboxes

The Edit Mode panel (below the canvas) has two checkboxes:

  • Anchor (default: on) — anchor points are selectable and moveable.
  • Control (default: on) — bezier control point handles are selectable and moveable.

Uncheck Control to ignore control points when clicking or rubber-banding, making it easier to select anchors in dense geometry.

Edge Selection Mode

Click the Edge Selection toolbar button. A Bézier segment (the curve between two consecutive anchor points) is treated as a single selectable edge.

Selecting Edges — Two-Step Scoping

  1. First click inside a polygon — scopes it (yellow dashed outline).
  2. Click within 15 px of a curve inside the scoped polygon — selects that edge.

Clicking directly on a curve in any polygon selects it and auto-scopes that polygon in one step.

Multiple Edge Selection

Click further edges to add them to the selection (toggle behaviour — clicking a selected edge deselects it). You can select edges from different polygons. When exactly two edges from different polygons meet the weld proximity criteria (≤80 px midpoint distance, directions within ~45°), they turn purple indicating they are weldable.

Operations in Edge Mode

  • Drag — moves the selected edge(s) and their adjacent control points.
  • Duplicate — creates an offset copy of the selected edge as a standalone single-edge polygon.
  • Delete — removes the selected curve segment from its parent polygon.
  • Weld (two edges selected, purple) — snaps both edges to their midpoint and registers weld links.
  • Extrude — ⇧ Shift + drag (see Operations tab).

Polygon Selection Mode

Click the Polygon Selection toolbar button. Entire polygons are selected and moved as units. This mode also works for discrete points created in Create Points mode — click near a point to select it.

Selecting Polygons

  • Click inside a polygon to select it. Its outline turns blue (Discrete) or orange (Relational).
  • Clicking a selected polygon again deselects it.
  • Click empty canvas to clear all selections.
  • ⌘ Cmd + click adds polygons to the selection without deselecting others.

Drag-to-Weld

While dragging a polygon in Polygon Selection Mode, the editor checks for nearby edges on unselected polygons. When edges meet the proximity and alignment thresholds, they highlight in purple. Release the mouse to snap and weld them automatically.

Discrete & Relational Sub-modes

Sub-modeWhat gets selectedWhat moves on drag
DiscreteOnly the directly clicked elementOnly that element — welded partners stay put. Useful for fine-tuning individual points without disturbing connected geometry.
Relational (default)Clicked element + all weld-linked elements (transitively)All linked elements together — shared seams remain intact. Essential when editing welded polygon meshes.

Switching Sub-mode

  • Tap ⇧ Shift while a selection mode is active → toggles between Discrete and Relational. Highlight colours update immediately.
  • ⇧ Shift + click the toolbar selection button when switching modes → activates directly in Discrete (default is Relational).
In Point Relational mode, clicking an anchor also selects its adjacent control points, keeping curve shape intact during moves. In Point Discrete mode, only the single clicked anchor or control point is selected.

Scoping

In Point and Edge Selection modes, a scope isolates one polygon so you can click precisely on its points or edges without hitting overlapping polygons from the same layer.

The scoped polygon is outlined with a yellow dashed border (2 px).

  1. Click inside a polygon → it becomes scoped.
  2. Click on a point or edge within the scoped polygon → it is selected.
  3. Click inside a different polygon → scope moves to it; prior selection is cleared (unless ⌘ Cmd held).
  4. Click empty canvas → scope and selection both cleared.

Marquee (Rubber-Band) Selection

Hold ⇧ Shift and drag on the canvas to draw a selection rectangle (shown in translucent blue).

  • Point mode — selects all visible points whose position falls inside the rectangle.
  • Edge mode — selects all edges whose anchor-midpoint falls inside the rectangle.
  • Polygon mode — selects all polygons whose centroid falls inside the rectangle.
When edges are already selected in Edge Selection Mode, ⇧ Shift + drag triggers Extrude instead of the marquee. Deselect all edges first (click empty canvas) if you want to marquee-select in edge mode.
