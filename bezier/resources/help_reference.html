Quick Reference

Keyboard Shortcuts

ShortcutAction
⌘ Cmd + ZUndo (up to 20 levels)
⌘ Cmd + CCopy selected polygons to clipboard
⌘ Cmd + VPaste polygons from clipboard (offset 20 px, on active layer)
⇧ Shift (tap, any selection mode active)Toggle current mode's sub-mode: Relational ↔ Discrete
⇧ Shift + click toolbar selection buttonActivate mode directly in Discrete sub-mode
⌘ Cmd + click point or edgeAdd to selection without deselecting others
⇧ Shift + drag (Edge mode, edges selected)Extrude selected edges
⇧ Shift + drag (any mode, no edges selected)Rubber-band marquee selection
