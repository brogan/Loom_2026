Quick Reference

Keyboard Shortcuts — Edit Menu (⌘ Cmd)

All Edit menu shortcuts fire from the menu bar accelerator. There are no duplicate panel key bindings, so each shortcut fires exactly once per keypress.

ShortcutAction
⌘ + ZUndo (up to 20 levels)
⌘ + ASelect All — selects all geometry of the current type on the active layer
⌘ + DDeselect All — clears all selections in every mode
⌘ + CCopy selected geometry to clipboard (leaves originals)
⌘ + VPaste clipboard at current mouse position (centred on cursor)
⌘ + XCut — copies selection to clipboard then deletes it

Pressure Sensitivity Summary

Pressure data flows from a stylus through Bezier Draw into the Loom rendering engine:

StageWhat happens
Bezier DrawStylus pressure at each anchor click is recorded per CubicPoint. Saved as pressure="0.xxx" attribute on anchor <point> elements in the openCurveSet XML. Anchor dot size in Point Selection Mode reflects recorded pressure.
Loom engine loadPressure values are read from the XML and stored on the Polygon2D as a pressures array (one entry per anchor).
Brush State initEach BrushEdge carries pressureStart and pressureEnd (the anchor pressures at its two endpoints).
Stamp enginePressure at each stamp position is linearly interpolated between pressureStart and pressureEnd. Applied as:
stampScale = baseScale × (1 − influence + pressure × influence)
Loom EditorSet Pressure → Size (0–1) and/or Pressure → Opacity (0–1) in the Brushes tab of a BRUSHED renderer. Values of 0 mean pressure has no effect; 1.0 gives full pressure control.
