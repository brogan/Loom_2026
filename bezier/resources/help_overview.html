Bezier Draw

Bezier Draw is a tool for creating the base geometry used by the Loom drawing engine. It produces three types of output:

  • Closed polygon sets — collections of closed cubic Bézier-curve polygons. These are the primary input for Loom's subdivision and rendering pipeline.
  • Open curve sets — non-closed splines with a distinct start and end. Used in Loom as stroked paths or brush-stamp paths; not subdivided.
  • Point sets — collections of discrete anchor positions. Used in Loom for scatter effects and brush-stamp placement; not subdivided.

Bezier Draw is normally launched from within the Loom Parameter Editor (via the Create / Edit buttons in the Geometry tab). When you close Bezier Draw, the editor automatically detects any new or changed files.

Top Panel (below toolbar)

  • Name field — the base filename used when saving. Spaces become underscores; non-alphanumeric characters are stripped.
  • Save button — writes all current geometry to the appropriate directories. Behaviour depends on content type (see View & Files tab).
  • Load button — opens a file browser. The file's root element determines what type is loaded (polygon set, open curve set, point set, or layer bundle).
  • SVG panel — Import SVG button for loading vector paths as polygon geometry.
  • Reference Image panel — browse for a background image to trace over.

Canvas (centre)

The drawing grid is 1000×1000 user units displayed on a 1040×1040 pixel panel (20 px border on each side). The grid has 10×10 major divisions. Coordinates are saved in a normalised form centred on 0 (typically −0.5 to +0.5) — the pixel-to-normalised conversion happens automatically on save.

Edit & Transform Panels (below canvas)

  • Edit Mode — Anchor and Control checkboxes control which point types are selectable/moveable in Point Selection mode.
  • Scale Axis — XY (proportional), X-only, or Y-only.
  • Rotation Axis — Local (each polygon's own centre), Common (shared centroid of selection), or Absolute (canvas centre).
  • Scale slider — drag to scale; release to commit and reset to zero. Operates on selection or all geometry if nothing is selected.
  • Rotation slider — drag to rotate (±100 → ±180°); release to commit and reset.

Creating Closed Polygons

Entering Create Polygons Mode

Click the Create Polygons toolbar button. The cursor enters drawing mode for a new polygon.

Placing Points

Each click on the canvas places an anchor point. After the first two anchors, smooth cubic Bézier control points are generated automatically between them. Continue clicking to add more segments.

Closing the Polygon

Click Close Polygon. The editor adds a final segment connecting the last anchor back to the first, records the polygon as finished, and switches to Polygon Selection Mode. A new empty polygon is ready for the next shape.

Closed polygons are required for Loom's subdivision pipeline. Use Close Polygon whenever you are creating shapes for subdivision rendering.

Creating Open Curves

Open curves are non-closed splines — paths with a distinct start and end. In Loom they are rendered directly as stroked paths or brush stamps (not subdivided).

  1. Click Create Polygons to enter drawing mode.
  2. Place anchor points as normal.
  3. When finished, click Finish Open Curve instead of Close Polygon. No closing segment is added — the path ends at the last anchor.
A canvas can contain a mix of closed polygons and open curves, but they are saved as separate files on disk. Keep them in separate files for clarity — close all shapes with Close Polygon first, then save; or use a dedicated session for open curves.

Creating Point Sets

Point sets are collections of discrete positions. In Loom they are rendered as dot stamps or brush stamps (not connected, not subdivided).

  1. Click Create Points to enter point-placement mode.
  2. Each click on the canvas places a magenta dot at that position.
  3. When finished, click Save. The points are written to pointSets/{name}_points.xml.
Points do not have bezier control points or edges. They are simply anchor positions. Selection modes still work — you can select, move, and delete points using Point Selection or Polygon Selection mode.

Layer System

All geometry is organised into named layers. Every polygon, open curve, and point belongs to exactly one layer.

Active Layer

The active layer (shown in bold in the Layers panel) receives all new geometry. Selection and hit-testing only operate on the active layer — you cannot accidentally select polygons on other layers. Click a row in the Layers panel to change the active layer.

Layer Visibility

  • Active layer — drawn at full opacity.
  • Other visible layers — drawn at 20% opacity (ghosted) as a reference.
  • Hidden layers (Vis unchecked) — not drawn at all.

Layer Buttons

Multi-layer Export

When multiple layers contain polygons, clicking Save writes a separate XML file per layer ({name}_{layerName}.xml) plus a {name}.layers.xml manifest. The Loom Parameter Editor can import the whole bundle as a layered polygon set. See the View & Files tab for full details.

Use layers to organise a complex polygon arrangement — for example, inner shapes on one layer and outer border shapes on another. Each layer can be given a different renderer in Loom.
