Bezier Editor

The Bezier Editor is a tool for creating and editing collections of closed cubic Bézier-curve polygons. Each polygon is built from one or more connected cubic Bézier segments. Finished polygon sets are saved as XML files that can be loaded directly into the Loom drawing engine.

Polygons can be freely transformed, welded together at shared edges, and extruded to build up complex multi-polygon shapes. The Weld Registry keeps track of which anchor points are linked across polygons so that Relational operations move welded neighbours together automatically.

Top Panel (below toolbar)

  • Load File / Save Polygons — browse for an existing XML polygon set or type a name and click Save to write one.
  • SVG — Import SVG button (see SVG tab).
  • Reference Image — browse for a background image to trace over.

Canvas (centre)

The drawing grid is 1000×1000 user units displayed on a 1040×1040 pixel panel (20 px border on each side). The grid has 10×10 major divisions. Click and drag directly in the canvas to create and edit polygons.

Edit & Transform Panels (below canvas)

  • Scale Axis — choose whether the Scale slider affects XY, X-only, or Y-only.
  • Rotation Axis — choose the pivot for the Rotate slider: Local (each polygon's own centre), Common (shared centroid of selection), or Absolute (canvas centre).
  • Scale slider — drag to scale; release to commit and reset to zero.
  • Rotation slider — drag to rotate (maps ±100 → ±180°); release to commit and reset.

Drawing Polygons

Entering Create Mode

Click the Create Polygon toolbar button (toggle). The cursor enters drawing mode for a new polygon.

Placing Points

Each click on the canvas places an anchor point. After the first two anchors the editor automatically creates smooth cubic Bézier control points between them. Continue clicking to add more segments to the current polygon.

Closing a Polygon

Click the Close Polygon button. The editor:

  1. Connects the last anchor back to the first, forming a closed loop.
  2. Records the polygon's centroid.
  3. Starts a fresh empty polygon manager for the next shape.
  4. Automatically switches to Polygon Selection Mode.
Every polygon in the Bezier Editor is a closed cubic Bézier path. Open paths are used internally for single-edge elements (connectors from extrude) but are not standalone shapes.
